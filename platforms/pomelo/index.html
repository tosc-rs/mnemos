A WebAssembly simulator for MnemOS

The left window shows a graphical forth shell. It is interactive when focused. The right window resembles a terminal into the system - tab completion and cursor up/down history is available for your convenience.

Type help for a list of commands, and consult the README for further details.

Note: Since currently no concept for EOF exists, cursor positioning may appear somewhat chaotic (we can't reset the prompt when some command finishes, because there is no "finish").
